Philosophy

SnakeJS is a lightweight library that allows you to write simpler and more readable JavaScript, and in any way you want, SnakeJS provides you with many facilities that other libraries provide in hundreds of unnecessary lines of code, which hurts the client when loading the page.

Facilities

This is one of the first versions of this library, whose version provides you with algorithms to facilitate process loads, manipulation of DOM elements, access Customer data more easily, Make requests to APIs, a Mini-Router, Form -Control, you just have to explore the library code :3.

Code

Being one of the trial versions, the effectiveness of the code cannot be guaranteed, nor that everything is in order, it is likely that many things will change, both variable names, eliminated methods etc..., but currently SnakeJS offers you a lot of functionalities and it saves you a lot of time, write the code as you see it.
